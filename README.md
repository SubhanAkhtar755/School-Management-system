## About

The **School Management System** allows admins to manage all aspects of a school‚Äôs operations in one place. Features include dynamic dashboards, live form controls, fee and result management, teacher tracking, and event management. Security is enforced with **keyboard-triggered admin access**, ensuring no unauthorized access.  

Students and staff data is synced in real-time using Firebase Realtime Database, providing instant updates across the platform.

---

## Features

- **Hidden Secure Admin Access:**  
  - Access via keyboard shortcuts only:  
    - `Shift + Ctrl : Login`  
    - `Shift + Ctrl : Register`  
    - `Shift + Ctrl : Go Back`  
  - Prevents random access to the system.  

- **Dynamic Admin Dashboard:**  
  - Total student & teacher count  
  - Real-time result tracking  
  - Class-wise breakdown for detailed insights  

- **Fee & Result Management:**  
  - Manage student fees, calculated per month  
  - Delete results via `Shift + D` + code entry  

- **Teacher Management:**  
  - Track teacher roles, subjects, and overall faculty stats  

- **Live Form Controls:**  
  - Admission & Job Application forms with toggles  
  - Admin can enable/disable any form dynamically  

- **Event System:**  
  - Create and manage school events from the panel  
  - No hardcoding needed  

- **Security:**  
  - Admin access protected with keyboard shortcuts  
  - Sensitive actions (like deleting results) require a code confirmation

## How It Works

1. **Admin Access:**  
   - Use `Shift + Ctrl` shortcuts to login, register, or navigate.  
   - Only authorized keyboard triggers unlock the system.  

2. **Managing Data:**  
   - Add/edit student and teacher information  
   - Manage fees and results in real-time  
   - Delete results securely using `Shift + D` + code  

3. **Forms & Events:**  
   - Enable/disable forms dynamically  
   - Manage school events without changing code  

4. **Real-Time Updates:**  
   - All changes sync instantly across the platform using Firebase Realtime Database
